<문제> 상수형 참조.가 기존의 참조 형식과 크게 다른 점이 무엇인지 답하세요.


상수형 참조는 처음 들어봤습니다만...
만약, 우측값(R Value) 참조 와 동일한 게 맞다면, 다음과 같습니다.
좌측값(식이 끝난 후 계속 존재하는 값)이 아닌 우측값(식이 끝나면 존재하지 않는 임시값)을 참조한다.

예:


int num = 10;
// 좌측값 참조
 int &lValue1 = num;
 // int &lValue2 = 20;      // 에러


// 우측값 참조
 int &&rValue1 = 20;
 // int &&rValue2 = num; // 에러
